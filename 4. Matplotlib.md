## Matplotlib
* it is for data visualization
* import matplotlib.pyplot as plt

## line plot
* plt.plot(x,y)
* plt.show()

## Scatter plot
* plt.scatter(x,y)
* plt.show()

## Labek
* plt.xlabel()
* plt.ylabel()

## Change axis scale
* plt.xscale('log')
