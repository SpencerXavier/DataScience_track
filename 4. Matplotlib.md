## Matplotlib
* it is for data visualization
* import matplotlib.pyplot as plt

## line plot
* plt.plot(x,y)
* plt.show()

## Scatter plot
* plt.scatter(x,y)
* plt.show()

## Label
* plt.xlabel("str")
* plt.ylabel("str")
* plt.title("str")

## Change axis scale
* plt.xscale('log')
* plt.xticks(list1,list2)...replace list1 with list2


## Note
